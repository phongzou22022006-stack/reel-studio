# Prompt Templates

Place your custom visual prompt templates here.

**Template format (one prompt per line or single multi-line template):**
```
Cinematic, 16:9, realistic photo, dramatic lighting, {scene_description}, shallow depth of field, color grading warm
```

**Available variables:**
- `{scene_description}` - What happens in the scene
- `{subject}` - Main subject
- `{style}` - Visual style override
- `{lighting}` - Lighting condition

**Examples:**
- `default.txt` - Generic cinematic style
- `hand_drawn.txt` - Rough doodle style (Zenn clone)
- `stock_footage.txt` - Search terms for Pexels/Pixabay

**Zenn Clone Style Example:**
```
Simple hand-drawn animated explainer, rough MS Paint / doodle style, dark blue-black background, warm orange firelight accents, minimal details, cinematic educational YouTube documentary, no text, no logos, 16:9, {scene_description}
```
