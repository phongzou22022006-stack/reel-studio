# Script Templates

Place your custom script templates here.

**Template format:**
```markdown
# {topic}

## Hook (0-3s)
{question}

## Problem (3-10s)
Most people think {common_misconception}, but actually {truth}.

## Twist (10-20s)
Here's what you're missing: {insight}.

## Solution (20-30s)
{actionable_tip}

## CTA (30-35s)
{call_to_action}
```

**Available variables:**
- `{topic}` - Video topic
- `{question}` - Hook question
- `{common_misconception}` - Common belief to challenge
- `{truth}` - The real insight
- `{insight}` - Key insight for twist
- `{actionable_tip}` - Actionable advice
- `{call_to_action}` - CTA text

**Examples:**
- `default.md` - Default viral hook template
- `zenn_style.md` - Zenn documentary style
- `short_form.md` - 15-second vertical format
