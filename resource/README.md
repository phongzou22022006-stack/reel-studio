# Resource Files

## Fonts
Place custom fonts here (`.ttf`, `.otf`)

Default: Uses system fonts
- Roboto-Bold.ttf
- Arial.ttf
- etc.

## Songs
Place background music here (`.mp3`, `.wav`)

Usage:
- Random mode: picks from this folder
- Custom mode: specify file name in config

**License:** Ensure you have rights to use any music files placed here.
